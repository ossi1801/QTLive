# QtLive

Minimal example on how to setup a QFileSystemWatcher to enable live coding in QML

![example](./docs/example.mp4)

## build


* Setup cmake: ```cmake -B build -S . -DCMAKE_BUILD_TYPE=Release```

* Compile: ```cmake --build build --config Release```

* Run: ```./build/QtLive```
